Description of the website:
A redone version of the Radio Pulze website (www.radiopulze.com). Radio Pulze is the official NUS campus radio station, completely run by NUS students.

Where to locate each element in your website:
GRID SYSTEM
- Grid system used throughout, but most noticeable in the home page.
RESPONSIVE DESIGN
- Responsiveness most noticeable in the navigation bar and home page.
BASE CSS
- Heading most noticeable in the services page (<h2> versus <p>).
- Blockquote in the testimonials in the services page.
- Buttons in the description and Contact Us form in the home page.
- Images are mostly circular (home page and services page) using img-circle.
COMPONENTS
- Glyphicons (play, stop - click to toggle) at the top left corner, in the navbar.
- Navbar at the top of every page.
- Jumbotron in every page, used as part of the header to contain the Radio Pulze logo.
- Well used to contain the Contact Us form at the bottom of the home page.
JAVASCRIPT
- Tooltips used in the Contact Us form at the bottom of the home page.
FORM VALIDATION
- Form validation (for emails without '@') at the bottom of the home page (Contact Us!).
JQUERY
- jQuery (play/stop button) in the navigation bar.

References you have taken:
- The current Radio Pulze website (www.radiopulze.com) for text and images (used with permission).
- Other images from Pixabay (no attribution required).
- w3schools html/css/bootstrap tutorials for the navigation bar and tooltips

Any special instruction to take note:
- The Submit button on the Contact Us form does not actually send any emails.
- The testimonials in the services page are made up, but based on true events.
- When using the play/stop button in the top left of the navigation bar, please press the icon instead of the text saying "TUNE IN!"
